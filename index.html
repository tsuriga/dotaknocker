Dota Knocker - The Dota API Query Tool

Search Dota 2 Web API for player and see statistics for their Heroes

Under the hood: Dota Knocker API queries the Dota 2 Web API for 25 previous public matches for a player and saves that data to a Firebase database. It then calculates and returns overall statistics for all Heroes that the player has used for any recorded matches found for the player.

Used tools

  • jQuery 2.1.0
  • Knockout 3.1.0
